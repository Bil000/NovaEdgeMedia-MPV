Need Help Getting Your API Credentials?
Google Ads Setup:
  1. Visit Google Ads API documentation
  2. Apply for a developer token
  3. Create OAuth2 credentials in Google Cloud Console
  4. Generate a refresh token using the OAuth2 flow
Meta Ads Setup:
  1. Go to Facebook Developers
  2. Create a new app and add Marketing API
  3. Generate an access token with ads_management permission
  4. Find your Ad Account ID in Meta Business Manager
